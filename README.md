# DayUI
UI components using <strong>Web Components</strong>

## Quick start

```html
<head>
    <script src="/dist/assets/index.js"></script>
    <link rel="stylesheet" href="/dist/assets/index.css">
</head>
<body>
<day-button>Button</day-button>
</body>
```
